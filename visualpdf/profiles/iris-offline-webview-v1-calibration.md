# Iris Offline WebView Visual Calibration v1

This profile definition is intentionally not a commercial-parity claim. The
qualified Apple and Android offline WebView renderer commands, device matrix,
comparison corpus identifiers, and reviewed pass/fail evidence must be recorded
before this profile is used for a release package.

The placeholder renderer path makes an unqualified invocation fail closed. The
zero tolerance values are not inferred defaults: they are explicit profile
values until the versioned calibration corpus supplies reviewed alternatives.
